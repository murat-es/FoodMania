**A restaurant-order android application by using Java.**

**Firebase is used as database.**

* Features:
  * Sign in/sign up
  * Night mode
  * English/Turkish language support
  * Update profile info
  * Forgot my password
  * Make reservation
  * Add to favorites
  * Add to cart
